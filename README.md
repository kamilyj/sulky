# sulky

SunEee frontend solution

## 安装使用

```js

npm install -g sulky

```


```js

$ sulky

  Usage: sulky <command>

  Commands:

    release     build and deploy your project
    server      launch a server

  Options:

    -h, --help     output usage information
    -v, --version  output the version number
    --no-color     disable colored output


```
